9  UAS-4: My Knowledge (Pengetahuan)

9.1 Pengenalan

Sebagai engineer sistem informasi, pengetahuan saya mencakup tidak hanya ide, tetapi juga struktur teknis untuk mewujudkannya. Pengetahuan dipetakan ke dalam tiga kategori:

9.2 Peta Pengetahuan Primitif (The “What”)

Fondasi pengetahuan deklaratif yang harus dikuasai:

9.2.1 Game Development Core

  • Pemahaman mendalam tentang Scene Tree, Signal, dan Physics Process di Godot Engine
  • Memahami game loop, event-driven programming, dan resource management

9.2.2 Linguistik Komputasional

  • Dasar-dasar pemrosesan teks dan suara manusia oleh komputer
  • Konsep Phoneme, Tokenization, NLP fundamentals
  • Speech recognition offline dan text-to-speech

9.2.3 Statistik Global

  • Data target audiens: wilayah buta huruf, penetrasi smartphone di negara berkembang
  • Demografi pengguna mobile entry-level, ketersediaan bandwidth

9.3 Peta Pemecahan Masalah (The “How”)

Pengetahuan prosedural tentang workflow pembuatan inovasi:

9.3.1 Fase Perancangan (Design)

  • Menerjemahkan kurikulum SD kelas 1 menjadi mekanik gameplay
  • Contoh: “Subjek-Predikat-Objek” → misi “Menyusun Jembatan Kata”
  • Game loop design, level progression, feedback system

9.3.2 Fase Optimasi (Optimization)

  • Memangkas penggunaan RAM agar berjalan lancar di perangkat 2GB
  • Pemrograman C++ tingkat rendah, memory profiling
  • Asset compression, batching, culling techniques

9.3.3 Fase Distribusi

  • Strategi penyebaran offline (peer-to-peer sharing) di desa tanpa internet
  • APK distribution, USB-based sharing, sync mechanisms

9.4 Artefak Pengetahuan

Bukti nyata kompetensi:

Artefak Deskripsi
GDD Cetak biru 50 halaman merinci setiap aspek permainan
Prototipe Alpha Versi playable dengan satu level tutorial
Video Devlog Dokumentasi visual proses pengembangan untuk komunitas

9.5 Sintesis: Mengintegrasikan Ketiga Pilar

  • What → Fondasi teoretis dan konseptual
  • How → Jalan eksekusi yang terukur
  • Artifacts → Bukti nyata dan terukur

Kombinasi ketiganya adalah satu-satunya cara untuk mewujudkan Aksara Quest yang berdampak sosial.
